# Sensor Module (Sensor Edition)

## Overview
The sensor module is intended to be removable and upgradeable. It collects posture and load metrics and streams them to the Coreline app.

## Data format (provisional)
- Timestamp, IMU (quat/accel/gyro), pressure array, battery level

## Privacy
- Raw sensor data stored on-device; user-controlled sharing to cloud
- Minimal personally identifiable information; encryption in transit
